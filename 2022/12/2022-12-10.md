## AdsignWeb構想にもとづいて開発環境を整える

### vite + html + tailwindcssを試してみる


#### 1. 作業記録

1. プロジェクトの作成

```bash
npm create vite@latest asw01 -- --template react
cd asw01
```

1. TailwindCSSのインストール

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
