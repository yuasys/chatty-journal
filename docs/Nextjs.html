この記事に沿って演習してみた

プロジェクトを作る

$ npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/master/basics/learn-starter"

nextjs-blog：任意のプロジェクト名

--use-npm：npmを使う宣言。yarnを使う場合は--use-yarn

--example：テンプレートとするソースコードを指定する

プロジェクトに入ってpackage.jsonを見る

{

  "private": true,

  "scripts": {

    "dev": "next dev",

    "build": "next build",

    "start": "next start"

  },

  "dependencies": {

    "next": "latest",

    "react": "17.0.2",

    "react-dom": "17.0.2"

  }

}

試しに、オプションを一切つけずにプロジェクトを作ってみたら、yarnが使われていた。デフォルトでは--user-yarnになっている。
